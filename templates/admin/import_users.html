Importer des utilisateurs

Le fichier doit être au format CSV avec les colonnes : Prénom, Nom, Email, Nom d'utilisateur, Rôles, Statut
Format du fichier CSV

Le fichier doit contenir les colonnes suivantes :

  • Prénom : Prénom de l'utilisateur
  • Nom : Nom de famille
  • Email : Adresse email (sera utilisée pour la connexion)
  • Nom d'utilisateur : Identifiant unique
  • Rôles : Rôles séparés par des virgules (ex: administrateur, gestionnaire)
  • Statut : Actif ou Inactif

Note : Le mot de passe par défaut sera "password123" pour tous les utilisateurs importés.
